# 소개

공공데이터 포털의 오픈API를 이용한 코로나 관련 통계와 같은 시각적 자료를 제공하는 사이트.

## 사이트 주소

https://project-cut.netlify.app

## 사용 스택

FrontEnd
=> React, styled-components,  axios

BackEnd
=> Express, request
